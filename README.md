## 🚀 Featured Projects

### 🏢 SocietyPay — AI-Powered Society Management Platform

> A full-stack mobile platform that automates residential society management using ML and NLP

**The Problem:**
- Complaints getting lost in WhatsApp groups
- Manual Excel sheets for payment tracking
- No way to predict who will default on maintenance

**What I Built:**

| Feature | Tech |
|---------|------|
| 📱 Mobile App | React Native + TypeScript |
| ⚙️ Backend API | Node.js + Express + JWT |
| 🗄️ Database | MongoDB + Mongoose |
| 🧠 NLP Complaint Routing | Python + Naive Bayes + CountVectorizer |
| 💰 Payment Risk Prediction | Scikit-learn + FastAPI |
| 💳 Payments | Razorpay + signature verification |
| 🛡️ Visitor Security | Isolation Forest anomaly detection |

**Results:**
- Complaints auto-categorized and routed instantly
- Payment defaulters flagged before they miss payment
- Visitor behavior analyzed in real time with ML
